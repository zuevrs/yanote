# yanote Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build `yanote` v1: service-side HTTP call recorder (Spring MVC) → `events.jsonl`, test tagging for RestAssured+Cucumber, and CLI to compute OpenAPI endpoint coverage + baseline/regression checks.

**Architecture:** A small multi-module Gradle build: shared core model/IO, Spring MVC starter that appends JSONL events, test libs for tagging headers, and a CLI that parses OpenAPI (via OSS parser), reads JSONL, computes coverage/uncovered and compares against baseline.

**Tech Stack:** Java 21 (or 17), Gradle Kotlin DSL, JUnit 5, Jackson (JSON + YAML), Picocli (CLI), `io.swagger.parser.v3:swagger-parser` (OpenAPI `$ref`), Spring Boot (starter + autoconfig), RestAssured, Cucumber JVM.
